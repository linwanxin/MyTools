# WeeklyPort
自动提交周报工具

## 使用方法：
#### 1.git下来
#### 2.注意boss.conf要和Weekport.py在一个目录下
#### 3.在boss.conf中编辑好要提交的内容；
#### 4.使用 python Weekport.py运行脚本即可；
